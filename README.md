## The Lingator

An accessibility focused application designed with non-native English speakers in mind. A large struggle for many is fully comprehending legal and medical documents. Lingator helps bridge this gap by breaking down complex and sometimes foreign language into clear, understandable explanations using AI-powered analysis.

## Features

- **Document Analysis** --> Upload legal or medical documents and receive easy-to-read breakdowns
- **Native-Language Summaries** --> Translates important takeways in the user's native language
- **Simple Interface** --> An intuitive, simple to understand interfact that alleviates pressure

## Languages

This application was made using JavaScript, TypeScript, and CSS.
